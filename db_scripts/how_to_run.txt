Step 1. Install nodejs, npm and postgresql.
Step 2. Run "npm install" in first the root directory (/) and then in the server directory (/server/);
Step 3. Use the database queries in /db_scripts/ directory to create and initialize a database.
Step 3. run command "npm run start" in the root directory, and in another terminal run "npm run dev" in the /server/ directory.

Database diagram is also there in /db_scripts/ directory
